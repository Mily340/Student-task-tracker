# Student Task Tracker Application

A Python application for students to manage their daily tasks using Object-Oriented Programming (OOP) principles.

## Features

- **Add Tasks**: Create new tasks with title and description
- **View Tasks**: Display all tasks in a formatted list
- **Update Tasks**: Modify existing tasks
- **Delete Tasks**: Remove tasks by ID
- **Search Tasks**: Find tasks by keyword
- **Persistent Storage**: Automatically saves tasks to JSON file
- **Error Handling**: Comprehensive error handling throughout
- **Task Statistics**: View task statistics

## Requirements

- Python 3.6 or higher
- No external dependencies required

## Installation

1. Clone or download the project files
2. Ensure all files are in the same directory:
   - `main.py`
   - `task.py`
   - `task_manager.py`
